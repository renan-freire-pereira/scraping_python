# RASPAGEM DE DADOS
A técnica de raspagem de dados é utilizada pelos programadores para realizar levantamento de dados em massa em diversos sites.
Neste repositório, pretende-se implementar o uso do scraping de duas maneiras distintas:
1. Pela biblioteca Request (publicado), pela qual se faz a raspagem no HTML estático da página;
2. Pela biblioteca Selenium, a qual é mais sofisticada e permite raspagem de dados pelas informações no DOM da página.
